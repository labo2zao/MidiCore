## Feature Overview

MidiCore provides professional MIDI performance features including:
- **Multi-track MIDI looper** with recording and playback
- **OLED user interface** with navigation and feedback
- **Footswitch control** for hands-free operation
- **Automation system** for complex sequences
- **Real-time MIDI processing** and routing
